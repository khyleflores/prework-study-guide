HTML

  • The head element contains information about the webpage.
  • Metadata not visible - accessibility, search engine and performance
  • The body element represents the visible content shown to the user.
  • Semantic HTML elemenents (name of the element descibes its content and role ie title, section
  • Child elements are elements inside other elements/Parent element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External CSS sheet is the most common way as it allows developers to keep all their CSS rules in a separate files.
  • Separation of Concerns
  • Do No Repeat Yourself (DRY) is a coding principle that encourages to reeuse or shaer piece of code to reduce the number of lines of codes
